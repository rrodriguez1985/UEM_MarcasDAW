Imágenes en HTML

En informática, podemos distinguir dos tipos de imagenes:
  • Imágenes de mapa de bits:Están formados por puntos de colores.
  • Imagénes vectoriales: Están formadas por figuras geométricas

Las ventajas que presentan unas con respecto a otras son las siguientes:
  • Ventaja de Img. Vectoriales: ocupan poco espacio y se pueden ampliar
    a cualquier escala sin que se pierda calidad de imagen.
  • Inconveniente de Img.Vectoriales: no permiten representar Imágenes
    sin formas definidas, como ocurren en las fotografías.
  • Ventajas de Img. Mapa de Bits: permiten representar imágnes sin formas
    definidas.
  • Inconveniente de Img. de Mapa de Bits: pierden mucha calidad al ampliarlas
    y ocupan mucho espacio en el disco duro.

Insertar Imagen en HTML

Para insertar una imagen en un documento HTML, debemos utilizar la monoetiqueta img seguido del atributo src (source)

Los atributos que podemos utilizar para la etiqueta img son los siguientes:
  • src= sirve para poder vincular la imagen con la etiqueta img.
    IMPORTANTE: recordar, que la carpeta img o los archivos de Imágenes
    que querais vincular, deben estar en la misma carpeta de trabajo.
  • alt= alternativo. Es un texto breve que muestra una descripción de la imagen
    cuando esta se "corrompe" o "está mal insertada".
  • height y width: para dar un valor de anchura y altura a las fotos.
    Esto se hará en CSS, con estos mismos atributos, pero no quita, que
    como caso ocasional, se pueda aplicar en HTML.
    Se pueden aplicar valores en PX (Pixel) o % (cogiendo el tanto % del
    tamaño que se tenga como original o de base de la imagen)
  • usemap: sirve para hacer mapeados de imágenes.
  • title= contiene texto a modo de tipo de información, que se muestra
    cuando se pasa el ratón por encima de la foto.

Etiqueta picture

La etiqueta picture, se utiliza sobre todo en páginas web responsive, puesto que
es un tipo de etiqueta que permite una mayor flexibilidad a la hora de utilizar
imágenes.
La etiqueta picture contine una o más etiquetas source, y una etiqueta img.
A su vez, va asociado a los siguientes atributos:
  • srcset: define la url de la imagen para mostrar.
  • media: acepta cualquier consulta de medios válida para definir un CSS Responsive.
  • width y height
  • type: define el tmipo de formato de imagen.

Etiqueta figure

La etiqueta figure se utiliza en HTML5 para mostrar un texto asociado a una imagen.
Su estructura está formada por:
  • etiqueta de apertura y cierre figure. Considerada como etiqueta padre
  • etiqueta img
  • etiqueta figcaption: sirve para establecer un título a una imagen.

Mapeado de Imagenes en HTML

En HTML, a través de la etiqueta map, podemos mapear una imagen, con el fin
de que podamos vincular varias partes o zonas de la misma a varias webs o
hiperenlaces distintos.
  • map: define la zona de mapeado que queremos realizar.
  • area: que sirve para establecer la zona de mapeado que queremos aplicar
  • shape: definir la zona y tipo de mapeado que queramos llevar a cabo
    • default: mapeado por defecto.
    • rect: mapeado en forma rectangular.
      Cogeremos como puntos de referencia vertice superior izquierdo de la
      imagen (X1,Y1), y la parte inferior derecha de la imagen (X2, Y2)
    • circle: mapeado de un área circular.
      Coger un punto X,Y que corresponderá al punto central de nuestra
      circunferencia, y un tamaño de radio (en px)
    • poly: mapeado de una zona con 3 o más puntos que definen un polígono.
      Cogeremos tres puntos o más (X1,Y1, X2,Y2, X3, Y3...) y estableceremos
      una zona de mapeado.
  • coords: coordenadas de la zona de mapeo.
