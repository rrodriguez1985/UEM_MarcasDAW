Listas en HTML

Las listas, son elementos que nos permiten organizar textos mediante un formato
más elegible y conciso.
En HTML, existen tres tipos de listas
  • Listas Ordenadas: vienen dadas por la etiqueta de apertura y cierre
    ol y /ol. ol significa (Ordered List)
  • Listas Desordenadas: vienen dadas por las etiquetas de apertura y cierre
    ul y /ul. ul significa (Unordered List)
  • Listas de Definición: vienen dadas por las etiquetas de apertura y cierre
    dl y /dl. dl significa Definition List.

Listas Ordenadas

La estructura de una lista ordenadas, ol, viene dada de la siguiente manera:
  • Tiene una etiqueta de apertura ol y una etiqueta de cierre /ol
  • Para poder definir el contenido de la lista, utilizaremos las etiquetas
    li y /li, que serán las que contendrán la información de nuestro listado.

Ejemplo de Lista Ordenada
  1. Kali Linux
  2. MS Windows 10/11
  3. Ubuntu
  4. MacOS
  5. Linux Mint

Listas Desordenadas

La estructura de una lista desordenadas, ul, viene dada de la siguiente manera:
  • Tiene una etiqueta de apertura ul y una etiqueta de cierre /ul
  • Para poder definir el contenido de la lista, utilizaremos las etiquetas
    li y /li, que serán las que contendrán la información de nuestro listado.

Ejemplo de Lista Desordenada
  • Programación
  • BBDD
  • SSII
  • ED
  • LMGSG
  • F.Cloud
  • Itinerario

Como podemos observar, la estructura de una lista ordenada y una lista
desordenada, es prácticamente la misma. Ambos tienen elementos li, y sólo
cambian la etiqueta de comienzo que tienen: ol / ul.
En base a esto, podemos establecer que la estiqueta ol y ul presentan una
serie de atributos que les permiten inicializar en un valor concreto a las listas ol
y cambiar la numeración por letras o números, y en el caso de las listas ul
podemos cambiar sólamente el ícono gráfico del lista.
Esto se realiza mediante el atributo type y start
Así, los tipos estructura que podemos utilizar en las listas son las siguientes:
  • Letras Mayúsculas o Minúsculas (Aa)
  • Cifras Romanas en Mayúsculas (I) o Minúsculas (i)
  • Número, por defecto sino ponemos nada, empezaríamos por el 1
  • disc (viñetas redondas rellenas)
  • circle (viñetas redondas huecas/vacias)
  • square (viñetas cuadradas rellenas)

Listas de Definición en HTML

Las listas de definición en HTML, tienen una estructura totalmente distinta
que las listas ol y ul.
  • Las etiquetas dl de apertura y cierre, que significan
    Description List
  • Las etiquetas dt de apertura y cierra, que significan
    Description Term
  • Las etiquetas dd de apertura y cierre, que significan
    Definition Description

Ejemplo de Lista de Definición
dl
Crea la caja que contiene los items de la lista
dt
Item que crea un término a definir
dd
Item que contiene una definición
