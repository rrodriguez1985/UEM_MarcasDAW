Propiedades en CSS

Cuando hablamos de propiedades en CSS, nos estamos refiriendo a las características
que podemos utilizar / aplicar a nuestras etiquetas HTML de manera generalizada.

Colores en CSS

Las propiedades que podemos utilizar en CSS para aplicar colores son las siguientes:
  • Aplicar mediante el atributo color el color de texto de un elemento.
  • Aplicar el atributo background-color para aplicar el color de fondo de un elemento.

Al igual que le podemos aplicar propiedades, podemos aplicarle distintas formas de

  • RGB Hexadecimal Abreviado: #f49

Unidades de medida en CSS

En CSS, podemos diferenciar entre dos tipos de medidas a utilizar:
  • Unidades Absolutas
  • Unidades Relativas

Unidades Absolutas

Las unidades absolutas, son aquellas que tienen un valor o tipo de medida fijo.
NO CAMBIAN NUNCA.
Son ideales para medios no variables, y por lo tanto son poco flexibles y no
son útiles de usar para la adaptación de diferentes resoluciones o pantallas.
  • Pulgadas (in)
  • cm
  • pc (Picas)
  • mm
  • pt (Puntos)

Unidades Relativas

Las unidades relativas, son un tipo de unidades de medida estandarizada en CSS,
qué dependen de otra resolución (densidad de pantalla).
Tienen un curva de aprendizaje más compleja, pero son ideales para trabajar
en dispositivos con distintas resoluciones y tamaños al ser muy flexibles.
  • em (1em = tamaño fuente (12-14px))
  • ex (aprox. 0,5 em)
  • rem (similar al em)
  • ch (zero width (1ch = ancho del cero))
  • px (pixel)
  • %

A parte de estas medidas, podemos utilizar las típicas como son las de width y height para dar
tamaño de anchura y altura de nuestras fuentes y elementos.

Modelos de Cajas en CSS

Todos los elementos HTML, cuando creamos páginas web, están dentro de lo que se conoce como contenedores / cajas de contenido.
Las cajas que vamos a utilizar, son principalmente div (cajas), o etiquetas semánticas
como pueden ser: header, nav, article, footer, etc.
Hasta ahora, utilizábamos tablas para maquetar, pero cuando utilizamos estas, teníamos la problemática
de que eran poco flexibles a la hora de poderlas ajustar a distintas resoluciones, y resultaban
muy complejas de manejar...
En la actualidad, cuando queremos maquetar vamos a dejar de usar tablas, y utilizaremos las cajas/divs que hemos comentado
junto a una serie de propiedades que comentaremos a continuación:
  • margin: corresponde al margen existente entre el borde exterior de nuestra caja/elemento
    contenedor, y nuestro navegador.
  • padding: corresponde al margen interior de nuestra caja, es decir, al espacio existente
    entre el contenido y el borde de nuestro elemento contenedor.
  • contenido: son las etiquetas que van ir dentro de nuestra caja o elelento contenedor.

Modelos de Cajas en CSS: Zondas de un elemento

Dentro de una caja, podemos encontrarnos las siguientes zonas a tener en cuenta:
  • Top: se refiere a la parte superior del elemento
  • Bottom: se refiere a la parte inferior del elemento
  • Right: se refiere a la parte dereha del elemento
  • Left: se refere a la parte izquierda del elemento
  • Center: que hace referencia a la posición central entre los elementos.
Estos elementos, los podemos aplicar de la siguiente manera:
  • margin-top / padding-top
  • margin-bottom / padding-bottom
  • margin-right / padding-right
  • margin-left / padding-left
La aplicación de estos elementos se pueden establecer de la siguiente manera:
  • margin con un solo valor: aplica a los 4 margenes de los elementos.
    P.E.: margin:40px;
  • margin con dos valores: el primer valor aplica a top y bottom, y el segundo
    aplica a right y left. P.E.margin:4px 5px;
  • margin con tres valores: el primero aplica a top,
    el segundo aplica a right y left, y el tercero, a bottom.
  • margin con cuatro valores: el primero aplica a top, segundo a right,
    tercero a bottom, y cuarto a left.
MUY IMPORTANTE: Cuando establecemos valores, siempre tenemos que tener en cuenta que
se seguirá el orden de las agujas del reloj para su aplicación.
Siendo en este caso, el primer valor top, segundo right, tercero bottom y cuarto
left. P.E. margin: 4px 5px 3px 8px.
Esto no quita, que podmos aplicar un valor a un lado concreto. Es decir, podríamos
establecer un margin/padding a una única zona.P.E. margin-top:20px.

Tipos de Bordes en CSS

CSS nos permite utilizar diversos tipos de propiedades para utilizar en los bordes de cajas,
imágenes, tablas, etc.
Para ello, podemos utulizar la siguiente propiedad:
  • border-color: sirve para poder dar color al borde
  • border-width: sirve para dar grosor al borde. Tipo: thin/medium/thick.
  • border-style: define el estilo del borde en CSS
Los tipos de bordes de border-style que podemos utilizar son los siguientes:
  • none: no aplica ningún tipo de estilo y coge el valor por defecto.
  • hidden: "ocultar" los bordes
  • dotted: establece puntos en los bordes
  • dashed: establece rayas discontinuas
  • solid: borde sólido
  • double: doble línea de borde
  • groove: biselado de luz desde arriba
  • ridge: biselado de luz desde abajo
  • inset: establece un borde con profundidad hacia dentro
  • outset: establece un borde con profundidad hacia fuera
