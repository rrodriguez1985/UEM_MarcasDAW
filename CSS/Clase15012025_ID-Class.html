Definición de ID en HTML y CSS

El valor del atributo "id" tanto en HTML como en CSS, hace referencia a un elemento que será único.
En HTML, cuando hago referencia a un id, utilizaremos el identificador y estructura
id = "nombre_identificador"
En CSS, para hacer referencia al ID que estamos atribuyendo ciertas propiedades y/o características,
lo haremos a través de la siguiente estructura: #nombre_idenficador
Así, podríamos decir que un elemento ID, será un elemento único.
Es decir, no deberá haber otro elemento con el mismo nombre de identificador (id) dentro de nuestra
página web / documento HTML.

Definición de CLASS en HTML y CSS

El valor del atributo "class" tanto en HTML como en CSS, hace referencia a varios elementos, independientemente
del tipo de etiqueta que sea, con el fin de poder aplicar unas mismas propiedades a estas.
Es decir, el valor de class puede ser utilizado en más de un elemento de un mismo documento HTML.
Esto lo que provoca es que podamos aplicar a distintas etiquetas un mismo estilo, ahorrando líneas de código
y sobre todo, poder aplicar una misma concordancia de estilos y estructura a nuestra web.
En HTML, cuando hago referencia a una clase, utilizaremos el identificador y estructura
class = "nombre_identificador"
En CSS, para hacer referencia al CLASS que estamos atribuyendo ciertas propiedades y/o características,
lo haremos a través de la siguiente estructura: .nombre_idenficador
