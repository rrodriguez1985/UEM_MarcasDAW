Definición de CSS

En las primeras páginas web, nosotros mezclamos lo que es contenido (html) con apariencia (estilos)
Desde hace unos, cuando creamos páginas web, lo hacemos de tal manera, que
lo que hacemos es tener separado la estructura (html) de la apariencia, y para ello
lo que hacemos es utilizar CSS.
CSS: se define o viene de la nomenclatura de Cascading StyleSheet (Hojas de Estilo en Cascada)
Una "hoja de estilo en cascada" se considera como etiquetas (marcas), que lo que nos permiten
es poder aplicar estilos, apariencias, colores, formas, etc. a nuestras páginas webs.
NO ES UN LENGUAJE DE PROGRAMACIÓN, SINO UN LENGUAJE DE MARCADO DE ESTILOS.

En resumen:
  • HTML define el contenido de mi página web: ¿qué es lo que hay?
  • CSS defino cómo será la apariencia de mi página web: ¿cómo es lo que hay?

Funcionamiento y Características

  • Las hojas de estilo CSS, son complementarias a las etiquetas HTML, y lo que hacen
    es mejorar la presentación y el aspecto de las páginas web.
  • Hay que separar el contenido de la apariencia a la hora de realizar la presentación de la webs
  • Al utilizar CSS, evitamos el uso excesivo de tablas en las páginas web
  • En CSS, como ocurría en HTML, no se indica en el código HTML el estilo del documento, sino que
    se indica lo que e cada elemento y dicho formato se especifica mediante CSS.
  • Todas las características que aplicamos de CSS nuestras web, son compatibles con HMTL4 y HTML5, así como posteriores.

En la actualidad, existen tres versiones de CSS, siendo la más estandarizada y utilizada la versión CSS3.

Ventajas de CSS

  • Mayor control de la presentación del sitio web
  • Si necesitamos poder modificar la presentación, podemos hacerlo de manera
    independiente sin tener que modificar la estructura HTML.
  • Mayor legibilidad --> resulta más fácil de interpretar y entender la web
  • Se reduce duplicidad de estilos
  • Es más fácil de crear versiones diferentes para diferentes dispositivos

Desventajas de CSS

  • A veces, dependiendo del navegador, la página que ha sido maquetada con CSS
    puede ser distinta e incompatible en cierta medida.
  • El uso de las tablas, nos permitía crear diseñoas complejos de forma mucho más
    sencilla que utilizando CSS.

Cómo usar CSS

Tenemos que tener en cuenta, que podemos usar CSS de tres maneras distintas:
  • CSS Externo: consiste en vincular a nuestro HTML, un documento externo con la extensión .css
  • CSS Interno: a través de etiquetas style (apertura y cierre), insertadas en el head del documento
  • CSS Inline/embebido/incrustrado: aplicamos el CSS de manera directa en la etiqueta mediante el atributo style
