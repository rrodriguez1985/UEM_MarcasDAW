Tablas en HTML

Las tablas en HTML, eran hasta hace unos años, un elemento fundamental para la maquetación
de nuestras páginas web.
Una tabla es un conjunto de celdas donde podemos alojar distintos contenidos.
La estructura de una tabla en HTML, está formada por los siguientes elementos:
  • table: Tiene etiqueta de apertura y cierre, y sirve para definir
    que vamos a utilizar una tabla en HTML
  • tr: Tiene etiqueta de apertura y cierre, y se utiliza para definir
    las filas de nuestra tabla.
  • td: Tiene etiqueta de apertura y cierre, y se utiliza para definr el número
    de celdas que va a tener nuestras filas.
  • Las celdas, pueden contener texto, imágenes, formularios, vídeos, audios, otras tablas...

Propiedades de Tablas en HTML

  • table:como inicio y fin de una tabla
  • caption: tiene etiqueta de apertura y cierra, sirve para poder
    poner un título a la tabla
  • align="top" o align="bottom": sirve para alinear arriva o abajo
    el título de la tabla
  • thead o th: Definir la cabecera de una tabla
  • tbody: sirve para definir el contenido de la tabla
  • tfoot: sirve para definir el pie de la tabla
  • tr: sirve para definir la fila de una tabla
  • td: sirve para definir la celda (columna) de la tabla

Combinación de tablas en HTML

Para poder realizar la combinación de celdas de tablas, podemos utilizar los siguientes
atributos:
  • colspan: indicar el número de filas que ocupa la celda (combinación de celdas)
  • rowspan: indicar el número de columnas que ocupa la celda (combinación de columnas)

Propiedades a tener en cuenta en tablas

Anteriormente, hemos visto algunas propiedades que forman parte de la estructura
de las tablas.
Ahora, veremos algunas propiedades que podemos usar y que debemos tener en cuenta
a la hora de tratar con tablas.
  • border: establece es el ancho del borde exterior de la tabla en px
  • cellspacing: especifica es el espacio entre las celdas
  • cellpadding: especifica el espacio entre el borde deuna celda y su contenido
  • width y height: podemos establecer un valor de anchura y altura a la tabla.
    Estos valores pueden ser en % y/o en px
